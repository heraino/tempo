Claude Code is
not Claude Chat

Claude Chat answers questions. Claude Code builds things — reads your files, writes code, runs commands, and ships to GitHub. Here's what that means in practice.

Section 01

Chat vs Code

Two very different tools. Toggle to see how each one works.

Claude Chat — Conversational AI
🗣️
You get answers Ask a question, get a text response. Great for explanations, drafting, brainstorming.
📋
Code is just text It shows you code in a chat bubble. You copy-paste it yourself.
🔁
No memory of your project Every session starts fresh. It can't see your files or what you built last time.
🚫
Can't take action It can't create files, run commands, commit to Git, or deploy anything.
Claude Code — Agentic Dev Tool
🏗️
You get results Describe what you want built. It writes the code, creates the files, done.
📂
Reads your entire codebase It understands your project structure, existing code, and conventions before touching anything.
🔄
Works across sessions Changes are committed to Git. Next session, the code is still there — your repo is the memory.
🚀
Takes real action Creates files, runs tests, commits code, pushes to GitHub, sets up deployments.
Section 02

What Claude Code can do

Things that are impossible or painfully slow in a chat window.

🔍

Understand any codebase

Point it at a project and ask "how does the auth system work?" — it reads the actual files and explains it precisely.

✏️

Edit files directly

No copy-paste. It opens the file, makes the change, saves it. Multi-file refactors in seconds.

🐛

Find and fix bugs

Show it an error message. It traces the root cause across files and fixes it — not just the symptom.

🔀

Commit and push to GitHub

Every change goes into Git with a meaningful commit message, pushed automatically. Full audit trail.

🌐

Set up deployments

Configure GitHub Pages, write CI/CD pipelines, or set up hosting — all from a plain English request.

🧪

Run and fix tests

Run your test suite, read the failures, fix the code, re-run. Loop until green — without you touching a terminal.

📦

Scaffold entire projects

Say "build me a React app with login and a dashboard." It creates the folder structure, all the files, wires it up.

🤔

Explain decisions

Ask "why is this written this way?" — it reads the context, the git history, and gives you a real answer.
